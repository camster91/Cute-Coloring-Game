# Calm Drawing - Feature Roadmap

A relaxing, mindful digital art application designed for teens, adults, and anyone seeking creative expression, stress relief, and artistic growth.

---

## Vision

Transform Calm Drawing from a children's coloring app into a sophisticated yet accessible creative tool that promotes mindfulness, artistic development, and relaxation for older audiences.

---

## Current Features (v2.0)

### Drawing Tools
- [x] Multiple brush types (pen, pencil, marker, watercolor, crayon, airbrush, calligraphy, highlighter, spray)
- [x] Pressure simulation based on drawing speed
- [x] Adjustable brush sizes and opacity
- [x] Eraser with variable sizes
- [x] Fill tool for areas

### Shape Tools
- [x] Rectangle, ellipse, line, triangle, star, arrow, hexagon, heart
- [x] Shift-constrain for perfect shapes
- [x] Fill and stroke options

### Canvas Features
- [x] Layers system with visibility, opacity, and lock controls
- [x] Symmetry modes (horizontal, vertical, quad, radial)
- [x] Grid overlay with snap-to-grid
- [x] Zoom (0.25x to 4x) and pan
- [x] Undo/Redo (50 states)

### Color System
- [x] Multiple color palettes
- [x] Hex color input
- [x] Color opacity control
- [x] Recent colors history

### Export & Save
- [x] PNG, JPEG, SVG export
- [x] Custom resolution settings
- [x] Transparent background option

### UI/UX
- [x] Dark/Light theme
- [x] Responsive design (desktop/tablet/mobile)
- [x] Keyboard shortcuts
- [x] Background music (6 ambient tracks)
